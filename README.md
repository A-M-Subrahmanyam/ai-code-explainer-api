# AI Code Explainer API

**🔗 Live API:** https://ai-code-explainer-api.onrender.com/explain  
**🔗 Frontend Repo:** https://github.com/A-M-Subrahmanyam/ai-code-explainer-ui  
**🔗 Live Demo:** https://ai-code-explainer-ui.vercel.app

Spring Boot backend that calls Google Gemini API to explain code.

## Tech Stack
- Java + Spring Boot + Render
- Google Gemini API
- CORS enabled for Vercel frontend

## API Endpoint
`POST /explain` - Send `{"code": "your code here"}`
